The Go programming language

PPRO 2014

Go, also called golang, is an open source programming language initially developed at Google in 2007

History

Robert Griesemer, Rob Pike and Ken Thompson started sketching a new language on September, 2007.

By January 2008, they started working on a compiler that generated C code as its output.

By mid-year the language had become a full-time project.

Go became a public open source project on November, 2009.

Motivation

Programming had become too difficult and the choice of languages was partly to blame

Programmers had to choose either:

  • efficient compilation
  • efficient execution
  • ease of programming

    • Go is an attempt to combine the ease of programming of an interpreted, dynamically typed language with the efficiency and safety of a statically typed, compiled language.
    It also aims to be modern, with support for networked and multicore computing, and at the same time is intended to be fast
